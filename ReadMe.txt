This is an application, determining is entered string a palindrome or not.

Applcation is using JavaFX FOR GUI, also includes unit tests using JUnit.

Author - Alina Goncarko

1) Building from source (running from palindrome-app folder):

  >mvn clean package

2) Running the application (from palindrome-app folder):

  >java -cp target/palindrome-app-1.0-SNAPSHOT.jar;lib/jfxrt.jar com.palindrome.app.Palindrome
